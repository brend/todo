# todo

A dead simple task list manager for the command line.

## Usage

Create a task:

``` sh
todo add -t "Buy milk"
```

List tasks:

``` sh
todo list
```

Mark a task as done by ID:

``` sh
todo complete -i 1
```

Delete a task by ID:

``` sh
todo remove -i 1
```

View help:

``` sh
todo -h
```
